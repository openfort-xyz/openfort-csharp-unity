# Openfort Unity Sample

This sample demonstrates:
- Authentication with Openfort
- Embedded wallet generation
- Signing messages
- Sending transactions

## Quick Start

```sh
pnpx gitpick openfort-xyz/openfort-csharp-unity/tree/main/sample/ openfort-embedded-wallet-unity && cd openfort-embedded-wallet-unity
```
